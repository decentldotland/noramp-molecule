## Build & Run


```bash

git pull https://github.com/decentldotland/noramp-molecule.git

```

Then setup your `.env` file variables with your [NoRamp](noramp.io) AppID and API key

```
NORAMP_APP_ID=...

NORAMP_API_KEY=...
```

Then run the following comand

```
npm install && npm run start
```
